# Portfolio & Firmenwebsite

Kombinierte Website zum Testen. Später einfach z.B. bei Netlify, GitHub Pages oder deinem Hoster hochladen.

## Lokal testen

1. Ordner `website` in Cursor/VS Code öffnen.
2. **Option A:** Datei `index.html` im Browser öffnen (Doppelklick oder Rechtsklick → „Öffnen mit“).
3. **Option B:** Mit Live-Server (Extension „Live Server“): Rechtsklick auf `index.html` → „Open with Live Server“.

## Sprachen

Die Website ist in **drei Sprachen** verfügbar: **Deutsch (DE)**, **Englisch (EN)** und **Serbisch (SR)**. Der Sprachumschalter (DE | EN | SR) steht in der Navigation und im Footer. Jede Sprache hat eigene Seiten (z. B. `index.html`, `index-en.html`, `index-sr.html`).

## Anpassen

- **Titel & Logo:** Der Name ist auf **ibkosanovic** gesetzt. Weitere Anpassungen in `index.html` bzw. den Sprachversionen.
- **Texte:** Alle Absätze in den Bereichen Über uns, Leistungen, Portfolio und Kontakt durch deine Inhalte ersetzen.
- **Kontakt:** E-Mail, Telefon und Adresse in der Sektion „Kontakt“ anpassen.
- **Portfolio:** Platzhalter „Projekt 1/2/3“ durch echte Projektbilder (z.B. `<img src="projekt1.jpg" alt="...">`) und Beschreibungen ersetzen.
- **Hintergrundbild:** Bild im Ordner `images` als **hero-bg.jpg** ablegen (Querformat, mind. 1200×600 px). Es erscheint im oberen Hero-Bereich.
- **Farben:** In `style.css` die Variablen unter `:root` anpassen (z.B. `--color-accent`).

## Veröffentlichen (später)

- Ordner `website` (mit allen Dateien) bei einem Anbieter wie **Netlify** oder **GitHub Pages** hochladen, oder
- Alle Dateien per FTP auf deinen Webspace legen.

Viel Erfolg!
